# Обзор OEM / SDK-интеграции

MYXON OEM SDK позволяет добавить возможности удалённого доступа в **ваше собственное железо** — без поставки отдельного шлюза. Если в вашем контроллере уже есть Linux-плата (Orange Pi, Raspberry Pi или подобная), вы можете установить агента MYXON рядом с вашим существующим ПО.

## Что вы получаете

- **Только исходящий туннель** — на объектах клиента не нужны правила firewall
- **Авто-обнаружение** — агент сам находит сервисы вашего контроллера в LAN
- **Одноразовая активация** — сгенерируйте код в Портале дилера, задайте его в env устройства — готово
- **Видимость парка** — все развёрнутые устройства видны из Портала дилера
- **Без пред-регистрации серийников** — сервер сам присваивает серийные номера при первой активации

## Требования

| Требование | Примечания |
|-------------|-------|
| Linux на базе Debian/Ubuntu | Проверено на Debian 11/12, Ubuntu 22.04 |
| Python 3.9+ | Обычно уже установлен |
| Исходящий интернет (порт 7000 TCP) | Для frpc-туннеля |
| Исходящий HTTPS (порт 443) | Для API регистрации агента |
| Архитектура | aarch64 (Orange Pi 5), armv7l, x86_64 |

## Чек-лист интеграции

- [ ] Получить дилерские креды у вашего аккаунт-менеджера MYXON
- [ ] Войти в Портал дилера
- [ ] Сгенерировать код активации для каждой партии устройств
- [ ] Запустить `install.sh` на каждом устройстве (или включить в образ ОС)
- [ ] Убедиться, что устройство появилось в Портал дилера → Устройства
- [ ] Завести аккаунт клиента и пригласить его

## Обзор архитектуры

```
  Ваша плата-контроллер (ARM Linux)
  ┌─────────────────────────────────────────────────┐
  │  Ваше ПО (Modbus-сервер, web UI, и т.д.)        │
  │  порт 80, 5843, 5900, ...                       │
  │                                                  │
  │  myxon_agent.py                                  │
  │    ├── скан LAN → находит порты выше             │
  │    ├── POST /api/v0/agent/activate               │
  │    └── frpc → туннель → MYXON Cloud              │
  └─────────────────────────────────────────────────┘
                         │ исходящий TCP :7000
                    ┌────▼─────────────────┐
                    │  MYXON Cloud         │
                    │  frps :7000          │
                    │  Backend :443        │
                    └────────────────────┘
```
