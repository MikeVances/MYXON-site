# Панель устройства

Панель устройства — главное рабочее место для одного устройства. Откройте её, кликнув по устройству в списке.

> **Скриншот:** *Шапка панели устройства: стрелка назад, имя «HOTRACO Orion — Коровник A», бейдж-щит с именем политики и кнопка обновления «5m».*

## Разделы панели

### VPN

Показывает текущий статус туннеля и базовую информацию об устройстве.

> **Скриншот:** *Карточка VPN: прогресс-бар (полный зелёный = онлайн), кнопка «Подключить VPN» (янтарная при онлайне), таблица со Статусом, Серийником, Прошивкой, Моделью, «Последним онлайном».*

| Поле | Описание |
|------|----------|
| Статус | `online` — туннель активен; `offline` — нет связи |
| Серийник | Серийный номер устройства |
| Прошивка | Версия прошивки агента (обновляется при регистрации) |
| Последний онлайн | Метка времени последнего heartbeat |

### Веб-доступ

Список всех сервисов, опубликованных устройством. Каждый сервис можно открыть как сессию удалённого доступа.

> **Скриншот:** *Карточка «Веб-доступ» с двумя ресурсами: «HOTRACO Remote+ (TCP · порт 5843)» и «VNC (VNC · порт 5900)». У каждого — кнопка «Подключиться» (янтарная) или метка «запрещено», если политика блокирует.*

::: info Ограничения политики доступа
Если ваша роль или политика доступа к площадке ограничивают тип ресурса, кнопка «Подключиться» показывается неактивной с меткой «запрещено». Обратитесь к админу для правки политик.
:::

**Открытие сессии:**
1. Нажмите **Подключиться** у нужного ресурса
2. Система создаёт сессию доступа с ограничением по времени (30 минут по умолчанию)
3. Нажмите **Открыть веб-доступ** в появившейся панели сессии

> **Скриншот:** *Панель активной сессии (голубой фон): «Активная сессия · TCP · remote-plus», с янтарной кнопкой «Открыть веб-доступ».*

### Прямой HMI

Для устройств HOTRACO (семейства Orion, Cygnus, Sirius) панель показывает **живой экран HMI** — тот же дисплей, что на физическом контроллере — с обновлением каждые 250 мс.

> **Скриншот:** *Раздел «Прямой HMI»: пиксель-в-пиксель дисплей контроллера с графиками температуры и настройками зон, в масштабе 3×.*

Это отображение экрана только для чтения. Интерактивное управление (нажатия клавиш) доступно через отдельные HMI-протокольные сессии.

### Статус облачной связи

Сводная панель со состоянием всех трёх облачных сервисов:

| Соединение | Что означает |
|------------|--------------|
| **VPN-соединение** | frpc-туннель — основной канал для сессий удалённого доступа |
| **Соединение конфигурации** | Связь с backend API для управления устройством |
| **Соединение логирования данных** | Метка времени последней синхронизации данных |

### Аварии

Живая панель аварий этого устройства, автообновление каждые 30 секунд.

→ [Документация по авариям](/ru/customer/alarms)

### Журнал событий

Аудит всех значимых событий устройства — регистрации, смены статуса, открытия/закрытия сессий.

| Столбец | Содержимое |
|---------|-----------|
| Кто | ID пользователя (первые 8 символов) или «система» |
| Когда | Локальная метка времени |
| Что | Тип события и детали |

> **Скриншот:** *Таблица журнала событий с 3 строками: «система» / «18.04.2026 14:23» / «Устройство активировано»; «система» / «18.04.2026 14:20» / «Агент зарегистрирован · tunnel_port:10001»; «система» / «18.04.2026 14:24» / «Сессия открыта · protocol:tcp».*
