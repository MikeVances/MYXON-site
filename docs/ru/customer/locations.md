# Локации

Локации (также **Площадки**) позволяют организовать устройства по физическому месту — коровник, теплица, здание. Так проще находить устройства и назначать разные политики доступа по локации.

## Страница «Локации»

Страница «Локации» — это стартовая после входа. Она показывает все ваши площадки с числом устройств у каждой.

> **Скриншот:** *Страница «Локации» со строкой поиска и двумя карточками площадок: «Коровник A» с иконкой, адресом «Фермерская 12, Дронтен», 3 устройства; «Теплица 1» с 1 устройством. Большая кнопка «+» для добавления локации.*

Кликните по карточке площадки, чтобы увидеть только устройства этой локации.

## Добавить локацию

1. Нажмите кнопку **+** на странице «Локации»
2. Введите имя площадки (обязательно)
3. Введите адрес (опционально — помогает опознать физическое место)
4. Сохраните

> **Скриншот:** *Диалог «Новая локация» с полем имени «Коровник A» и адреса «Фермерская 12, Дронтен, Нидерланды». Кнопки «Отмена» и «Сохранить».*

## Привязать устройство к локации

Привязать устройство к площадке можно двумя способами:

**При активации устройства** (шаг 3 мастера):
- Выберите площадку из списка при активации нового устройства

**С панели устройства**:
- Откройте устройство → настройки → сменить привязку к площадке

## Непривязанные устройства

Устройства без привязки к площадке отображаются в разделе **«Без привязки»** в списке. Используйте фильтр в сайдбаре `?site=unassigned`, чтобы их найти.

## Контроль доступа по локации

Политики доступа на уровне площадки позволяют админам точно управлять тем, что может каждый пользователь в каждой локации. Это полезно, когда за разные здания отвечают разные сотрудники.
