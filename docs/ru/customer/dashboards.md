# Дашборды

Соберите свои страницы обзора из **карточек** — живых значений, графиков трендов, картинок и заметок, разложенных как вам удобно. Выведите на один экран те цифры, что важны для вашего производства.

> **Скриншот:** *Дашборд «Обзор теплицы» с сеткой карточек: крупное значение температуры, спарклайн влажности, картинка плана и текстовая заметка.*

## Типы карточек

| Карточка | Что показывает |
|----------|----------------|
| **Значение** | Одно число из метрики устройства — последнее показание или агрегат (среднее, мин, макс) за период |
| **Тренд** | Спарклайн метрики во времени |
| **Картинка** | Любое изображение по URL — план, фото, схема |
| **Заметка** | Свободный текст — подписи, инструкции, напоминания |

Карточки значения и тренда берут данные из [метрик устройства](/ru/customer/device-dashboard) (логирование должно быть включено на устройстве).

## Как собрать дашборд

1. Откройте **Дашборды** в левом меню.
2. Нажмите **Новый** и дайте дашборду имя.
3. Нажмите **Редактировать**, чтобы войти в режим правки.
4. Кнопкой **Добавить карточку** выложите нужные типы.
5. **Перетаскивайте** карточки, меняйте размер за угол.
6. Нажмите **шестерёнку** на карточке, чтобы настроить её (выбрать устройство + метрику для значения/тренда, URL для картинки, текст для заметки).
7. Нажмите **Сохранить**.

> **Скриншот:** *Дашборд в режиме правки: сверху палитра кнопок «Добавить карточку», у карточек — маркеры перетаскивания и наложение с шестерёнкой/удалением.*

## Метрики доступности

Карточка значения может показывать **uptime %** — долю времени, что параметр оставался в заданном пороге за период. Это простой индикатор состояния оборудования с одного взгляда (прокси доступности).

## Кто может редактировать

Смотреть дашборды может любой в вашей организации. **Редактировать** (создавать, менять раскладку, удалять) — только роли админа. Дашборды относятся к вашей организации и могут быть привязаны к конкретной площадке или устройству.
